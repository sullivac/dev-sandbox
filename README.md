# dev-sandbox

A Docker container image for use with Visual Studio Code built on top of Ubuntu 18.04 (Bionic Beaver).

It supports:

* Node.js
* Python
* PowerShell Corey
* .NET
* Java
* Docker CLI
* Kubectl

It contains the volumes for:

* Main sandbox
* Visual Studio Code extensions
* PowerShell profiles
* PowerShell environment (modules)
